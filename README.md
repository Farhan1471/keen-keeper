# KeepKeeper
Keen Keeper is a simple app made with React that helps to stay in touch with friends. You can see when you last talked to someone and easily save things like calls, texts, or video chats. It has a clean dashboard, separate pages for each friend, a timeline to view past interactions, and simple stats that shows interaction activity in visual way.

# Technology Used
1. React.Js
2. React Router
3. Tailwind CSS
4. DaisyUI
5. React Toastify
6. React Icons
7. React Spinners
8. Recharts

# Key Features
1. Quick check-in actions to log Call, Text, and Video interactions.
2. A timeline to see previous interactions.
3. A Pie-chart is showing activity.
